# BYOB Ingestion Sample Documents

This folder contains small documents for testing BYOB upload, parsing, chunking, embedding, preview rendering, and retrieval.

Suggested upload order:

1. `01_mixed_markdown.md`
2. `02_plain_text.txt`
3. `03_preview_html.html`
4. `04_structured_word.docx`
5. `05_layout_image_pdf.pdf`

The samples intentionally include headings, lists, tables, formulas, long paragraphs, English and Chinese text, and image-like content.
